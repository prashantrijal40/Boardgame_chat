# Boardgame_chat

Instructions necessary to run application
As a prerequisite, install python 3.8 or higher and setting up virtual environment (venv) and SQLite which doesn’t requires installation as it is used by default.
Setup Instructions
•	Extract the file and make sure it’s in the root directory.
•	Navigate to the project directory
cd boardgame_chat
•	Create a virtual environment using python
python3 -m venv venv
source venv/bin/activate  # On macOS/Linux
venv\Scripts\activate     # On Windows
•	Install required packages by pip
pip install -r requirements.txt
•	And finally run the application using 
python run.py
